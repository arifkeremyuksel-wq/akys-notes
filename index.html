☁️ Bulut Senkronu Kur
Supabase bilgilerini girerek tüm cihazlarda gerçek zamanlı senkron aktifleştir. Ücretsiz.
1 Supabase'de proje aç
supabase.com → New Project → Proje oluşturduktan sonra Settings › API'den Project URL ve anon public key'i kopyala.
2 SQL ile tabloları oluştur
Supabase SQL Editor'a git ve aşağıdaki komutu çalıştır:
-- AKY's Notes tabloları create table if not exists fn_notes ( id text primary key, entity_id text not null, date text, contact text, summary text, followup text, extra text, actions jsonb default '[]', updated_at timestamptz default now(), created_at timestamptz default now() ); create table if not exists fn_entities ( id text primary key, folder_id text not null, name text not null, updated_at timestamptz default now() ); create table if not exists fn_folders ( id text primary key, name text not null, icon text default '📁', updated_at timestamptz default now() ); -- RLS: sadece sen erişebilsin (opsiyonel ama önerilir) alter table fn_notes enable row level security; alter table fn_entities enable row level security; alter table fn_folders enable row level security;
